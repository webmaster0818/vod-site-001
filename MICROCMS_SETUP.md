# microCMS設定ガイド

このガイドでは、microCMSの初期設定から記事投稿までの手順を説明します。

## Step 1: microCMSアカウント作成

1. **microCMS公式サイトにアクセス**
   - https://microcms.io/

2. **無料アカウント登録**
   - 「無料で始める」をクリック
   - メールアドレスまたはGoogleアカウントで登録

3. **サービス作成**
   - サービス名: `vod-site-minori`（任意）
   - サービスID: `vod-site-minori`（英数字とハイフン、後で変更不可）
   - プラン: **Hobbyプラン（無料）**

## Step 2: API作成

1. **ダッシュボード → 「API作成」**

2. **基本設定**
   - API名: `posts`
   - エンドポイント: `posts`
   - APIの型: **リスト形式**

3. **「作成する」をクリック**

## Step 3: フィールド設定

「posts」API → スキーマ設定 → フィールドを追加

### 1. title（タイトル）

- フィールドID: `title`
- 表示名: タイトル
- 種類: テキストフィールド
- 必須: ✅

### 2. description（説明）

- フィールドID: `description`
- 表示名: 説明
- 種類: テキストエリア
- 必須: ✅
- 説明: 120文字以内で作品の簡潔な説明

### 3. category（カテゴリ）

- フィールドID: `category`
- 表示名: カテゴリ
- 種類: セレクトフィールド
- 必須: ✅
- 選択肢:
  - 映画
  - ドラマ
  - アニメ
  - ライブ
  - スポーツ

### 4. genre（ジャンル）

- フィールドID: `genre`
- 表示名: ジャンル
- 種類: テキストフィールド
- 必須: ✅
- 説明: アクション、コメディ等

### 5. releaseYear（公開年）

- フィールドID: `releaseYear`
- 表示名: 公開年
- 種類: 数値
- 説明: 2024等

### 6. rating（評価）

- フィールドID: `rating`
- 表示名: 評価
- 種類: 数値
- 最小値: 0
- 最大値: 5
- 説明: 0.0〜5.0の評価

### 7. thumbnail（サムネイル）

- フィールドID: `thumbnail`
- 表示名: サムネイル
- 種類: 画像

### 8. content（本文）

- フィールドID: `content`
- 表示名: 本文
- 種類: リッチエディタ
- 必須: ✅

### 9. vodServices（VODサービス）

- フィールドID: `vodServices`
- 表示名: VODサービス
- 種類: 繰り返しフィールド

**繰り返しフィールド内のフィールド:**

#### 9-1. name（サービス名）
- フィールドID: `name`
- 表示名: サービス名
- 種類: テキストフィールド
- 説明: Netflix、Amazon Prime Video等

#### 9-2. url（URL）
- フィールドID: `url`
- 表示名: URL
- 種類: テキストフィールド
- 説明: サービスのURL

#### 9-3. available（配信中）
- フィールドID: `available`
- 表示名: 配信中
- 種類: 真偽値

### 10. tags（タグ）

- フィールドID: `tags`
- 表示名: タグ
- 種類: 複数選択
- 選択肢: （後で自由に追加可能）
  - アクション
  - コメディ
  - ドラマ
  - サスペンス
  - 洋画
  - 邦画
  - アニメ
  - SF
  - ホラー
  - ファンタジー
  - 家族向け
  - 大人向け
  - 感動
  - 笑える
  - 泣ける

### 11. publishedAt（公開日）

- フィールドID: `publishedAt`
- 表示名: 公開日
- 種類: 日時

## Step 4: APIキー取得

1. **サービス設定 → APIキー**

2. **「X-MICROCMS-API-KEY」をコピー**

3. **Next.jsプロジェクトの `.env.local` に設定**

```env
MICROCMS_SERVICE_DOMAIN=your-service-domain
MICROCMS_API_KEY=xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
```

## Step 5: テスト記事作成

1. **「posts」API → 「コンテンツを追加」**

2. **各フィールドを入力**

**例:**

- タイトル: テスト記事
- 説明: これはテスト記事です
- カテゴリ: 映画
- ジャンル: アクション
- 公開年: 2024
- 評価: 4.5
- 本文: これはテスト記事の本文です。
- VODサービス:
  - name: Netflix
  - url: https://www.netflix.com
  - available: ✅
- タグ: アクション、洋画
- 公開日: （今日の日付）

3. **「公開」をクリック**

4. **サイトで確認**
   - http://localhost:3000 にアクセス
   - 記事が表示されることを確認

## Step 6: 記事作成フロー（運用）

### 新規記事作成

1. microCMS管理画面にログイン
2. 「posts」→ 「コンテンツを追加」
3. 各フィールドを入力
4. 「公開」をクリック
5. 60秒以内にサイトに反映

### 記事編集

1. microCMS管理画面にログイン
2. 「posts」→ 編集したい記事をクリック
3. 内容を修正
4. 「更新」をクリック
5. 60秒以内にサイトに反映

### 記事削除

1. microCMS管理画面にログイン
2. 「posts」→ 削除したい記事の「...」→ 「削除」
3. 確認ダイアログで「削除」
4. 60秒以内にサイトから消える

## よくある質問

### Q1. 画像はどこから取得すればいい？

**A:** Unsplash（https://unsplash.com/）がおすすめです。

1. Unsplashで作品名やキーワードで検索
2. 気に入った画像を選択
3. 「Download free」→ 「Small」
4. microCMSの「サムネイル」フィールドにアップロード

### Q2. リッチエディタの使い方は？

**A:** Wordのような感覚で編集できます。

- **見出し:** H2, H3を使用（H1は使わない）
- **太字:** テキストを選択 → Bボタン
- **リスト:** テキストを選択 → リストボタン
- **リンク:** テキストを選択 → リンクボタン
- **画像:** 画像ボタン → アップロード

### Q3. 複数のVODサービスを追加するには？

**A:** 「vodServices」フィールドで「+ 追加」を何度もクリックします。

例:
- Netflix（配信中）
- Amazon Prime Video（配信中）
- Disney+（配信なし）

### Q4. タグはどう選ぶ？

**A:** 作品の特徴を3-5個選びます。

例（アクション映画）:
- アクション
- サスペンス
- 洋画
- 笑える
- 家族向け

### Q5. 記事が反映されない

**A:** 以下を確認してください:

1. 記事を「公開」したか（下書きのままではない）
2. 60秒待った（ISRのキャッシュ更新）
3. ブラウザをリロードした
4. `.env.local` が正しく設定されているか

## トラブルシューティング

### エラー: microCMS API Error

**原因:**
- APIキーが間違っている
- サービスドメインが間違っている
- APIが作成されていない

**解決:**
1. `.env.local` を確認
2. microCMSダッシュボードでAPIキーを確認
3. サービスドメインを確認（例: `vod-site-minori`）

### 記事一覧が空

**原因:**
- 記事を公開していない（下書き状態）
- microCMSに記事が1件も登録されていない

**解決:**
1. microCMS管理画面で記事を確認
2. 「公開」状態になっているか確認
3. テスト記事を作成して確認
